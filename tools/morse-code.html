Text to Morse Code

Instantly translate text to Morse code and vice versa.

About this Tool

Our Text to Morse Code translator allows you to convert standard English characters and numbers into international Morse code dots and dashes, or perform the exact opposite. It is a handy tool for hobbyists, ham radio operators, and puzzle enthusiasts.

How to Use

  1. Paste your English text or Morse code into the top input box.
  2. Click "Encode Text" to translate letters and numbers into dots and dashes.
  3. Alternatively, click "Decode Morse" to turn dots (`.`) and dashes (`-`) back into readable text.
  4. Click "Copy" to save your result.

Frequently Asked Questions

What format of Morse code is accepted?

The decoder expects standard dots (`.`) and dashes (`-`). Letters should be separated by a single space, and words should be separated by a slash (`/`) or three spaces.

Are special characters supported?

Standard punctuation marks like periods, commas, and question marks have corresponding Morse code symbols and are supported. Other special symbols will be ignored.

Does case matter?

No, Morse code does not distinguish between uppercase and lowercase letters. The encoder will treat everything uniformly.
